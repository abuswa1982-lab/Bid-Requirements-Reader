Bid Kickoff Sheet

This tab doesn't read any documents itself. It reads what the other tabs already found — the spec extraction, Checklist 1, Checklist 2, the drawing extraction, and Checklist 3 — and weighs them together as one project, not five separate lists. It's looking for the things that only show up when you compare documents against each other: a spec requirement the drawings contradict, a drawing flag that changes how risky a "Covered" checklist item really is, that kind of thing.

How to read this: every requirement carries a source stamp (§) pointing to the spec section or page it came from. A ? stamp means the tool could not pin the exact location, so check that one yourself. A VERIFY flag means the tool was not fully sure the requirement applies. The GREEN/YELLOW/RED read and the four risk scores are an advisory flag, not a verdict, confirm before pricing. The tool prepares and points. You approve.

Built to be calibrated: run it on a bid you already closed, compare against what you know was required, and the gaps show exactly what to tune. For this to catch insurance, bonding, and contract risk correctly, make sure the Agreement and General/Supplementary Conditions sections are included in what you upload, not just the Bid Form and Instructions to Bidders.
